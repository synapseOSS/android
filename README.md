# Synapse Social Android

Source code for the Synapse Social Android application.

## About

Synapse Social is an Android social networking application built with Kotlin and Jetpack Compose.

**Version:** 1.0.0-alpha07  
**Package:** com.synapse.social.studioasinc

## Technology Stack

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Minimum SDK:** 26 (Android 8.0)
- **Target SDK:** 34 (Android 14)
- **Compile SDK:** 36
