## üõ†Ô∏è Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/) (2.2.21)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/pack/compose) with Material 3
- **Backend**: [Supabase](https://supabase.com/) (Auth, Database, Realtime, Storage, Functions)
- **Dependency Injection**: [Hilt](https://dagger.dev/hilt/)
- **Image Loading**: [Coil](https://coil-kt.github.io/coil/) & [Glide](https://github.com/bumptech/glide)
- **Networking**: [OkHttp](https://square.github.io/okhttp/) & [Ktor](https://ktor.io/)
- **Database**: [Room](https://developer.android.com/training/data-storage/room) (Local Caching)
- **Architecture**: MVVM (Model-View-ViewModel)
