**Synapse Social** is a high-performance, modern social media platform built for Android. Leveraging the power of Jetpack Compose for a fluid UI and Supabase for a robust backend, it offers a seamless experience for real-time interaction and media sharing.

---

## ✨ Features

- **🚀 Real-time Communication**: Instant messaging and updates powered by Supabase Realtime.
- **🎨 Modern UI/UX**: Built entirely with Jetpack Compose following Material 3 guidelines for a premium feel.
- **🔐 Secure Authentication**: Multi-factor ready auth via Supabase Auth, including biometric support.
- **📸 Rich Media Sharing**: Support for high-quality images and video uploads with multi-provider integration (Supabase Storage, Cloudinary, ImgBB).
- **🔔 Smart Notifications**: Real-time push notifications via OneSignal.
- **🛡️ Privacy Focused**: Granular settings for profile visibility and data management.

---

## 🛠️ Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/) (2.2.21)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/pack/compose) with Material 3
- **Backend**: [Supabase](https://supabase.com/) (Auth, Database, Realtime, Storage, Functions)
- **Dependency Injection**: [Hilt](https://dagger.dev/hilt/)
- **Image Loading**: [Coil](https://coil-kt.github.io/coil/) & [Glide](https://github.com/bumptech/glide)
- **Networking**: [OkHttp](https://square.github.io/okhttp/) & [Ktor](https://ktor.io/)
- **Database**: [Room](https://developer.android.com/training/data-storage/room) (Local Caching)
- **Architecture**: MVVM (Model-View-ViewModel)
